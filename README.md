### 👋 About me

I'm a fourth-year Computer Science student specializing in **Software Engineering**. I like
building software that genuinely **helps or entertains** — from polished web apps to playful
browser audio toys and the tooling I use to build them faster.

- 🔭 Currently building **THUMP**, a browser groovebox, and improving my portfolio.
- 🧠 Into clean architecture, opinionated design, and automating the boring parts.
- 🌱 Exploring AI agent orchestration and Cloudflare Workers.
- 💬 Ask me about React, TypeScript, or building audio in the browser.
